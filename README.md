### Prerequisites

### Installation

```bash
# Install dependencies
npm install --legacy-peer-deps

# Copy config example
cp src/config.example.ts src/config.ts

# Add your OpenAI API key to src/config.ts

# Start the app
npm start
```

### Running on Different Platforms

```bash
# Web
npm run web

# iOS (requires Mac)
npm run ios

# Android
npm run android
```
